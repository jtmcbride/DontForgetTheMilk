# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Tasks

- `GET /api/tasks`
  - All user tasks and search
- `POST /api/tasks`
- `GET /api/tasks/:id`
- `PATCH /api/tasks/:id`
- `DELETE /api/tasks/:id`

### lists

- `GET /api/lists`
- `POST /api/lists`
- `GET /api/lists/:id`
- `DELETE /api/lists/:id`
- `GET /api/lists/:id/tasks`
